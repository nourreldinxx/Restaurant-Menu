## Pages
- **Home** - Hero section, menu categories, about us, events, reviews
- **Menu** - Filterable food items by category
- **About** - Restaurant story, features, statistics, testimonials
- **Booking** - Table reservation form
- **Contact** - Contact form

## Features
- Responsive design
- Menu filtering system
- Component-based structure
- Modern UI with Google Fonts & Font Awesome

## Setup
1. Open `index.html` in a browser
2. Components load automatically via JavaScript
3. Navigate between pages using header

## Tech
HTML5, CSS3, JavaScript, Google Fonts, Font Awesome
